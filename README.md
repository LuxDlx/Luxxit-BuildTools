# Luxxit - BuildTools

# How to build?

## Prequisites
- Python 3.9 or newer
- git

## Build Instructions (Linux/Windows)
> ``git clone https://github.com/LuxDlx/Luxxit-BuildTools``
> 
> ``cd Luxxit-BuildTools``
> 
> ``python main.py``

## How to run after building
> Windows: ``./luxxit``
>
> Linux: ``chmod +x ./luxxit.sh && ./luxxit.sh``
